# 🔤 Strings in Python

Welcome to the **Strings** section of the **Python Basics** repository.

Strings are one of the most commonly used data types in Python. They are used to store and manipulate text such as names, email addresses, passwords, messages, file paths, and much more.

In this section, you'll learn how to create strings, work with indexing and slicing, use built-in string methods, format strings efficiently, and perform pattern matching using Regular Expressions (Regex).
