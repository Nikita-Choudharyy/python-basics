# Python Functions

This folder covers the fundamentals of reusable code blocks in Python, moving from basic definitions to functional programming tools like Lambda, Map, Filter, and Reduce.

## 📌 Topics Covered

- **User-Defined Functions:** Arguments, parameters, return values, and default parameters.
- **Anonymous Functions:** Single-line `lambda` expressions for quick operations.
- **Functional Tools:** - `map()`: Transforming iterables element-wise.
  - `filter()`: Selecting elements based on conditions.
  - `reduce()`: Accumulating elements into a single value.

## 🚀 Data Science Context

Functions are the building blocks of clean data pipelines. In AI/ML, we heavily use `map()` and `apply()` with Pandas dataframes to clean data columns efficiently without writing messy `for` loops.
