# 📦 Data Structures in Python

This folder contains beginner-friendly Python programs that explain the core data structures in Python through simple examples and hands-on practice.

## 📚 Topics Covered

* **Lists** – Creating lists, indexing, slicing, updating elements, and common list methods.
* **Tuples** – Creating tuples, unpacking, indexing, slicing, and tuple methods.
* **Sets** – Creating sets, adding/removing elements, set operations, and unique values.
* **Dictionaries** – Working with key-value pairs, dictionary methods, updating data, and looping through dictionaries.

## 🎯 Learning Objectives

By completing the programs in this folder, you will learn how to:

* Create and use Python data structures.
* Access and modify stored data.
* Apply built-in methods for each data structure.
* Understand the differences between Lists, Tuples, Sets, and Dictionaries.
* Choose the appropriate data structure for different programming tasks.
