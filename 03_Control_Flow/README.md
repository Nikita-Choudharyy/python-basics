# 🔀 Control Flow in Python

Welcome to the **Control Flow** section of this repository.

Control Flow determines the order in which a program executes its statements. By using conditional statements, Python programs can make decisions, execute different blocks of code, and solve real-world problems based on specific conditions.

This section introduces the fundamentals of decision-making through **Jupyter Notebooks** for detailed learning and **Python Source Code** for quick revision and practice.

---

## 📚 Topics Covered

* 🔀 Conditional Statements
* 🎯 Match-Case Statement
* 🪆 Nested Conditions

---

## 🎯 Learning Objectives

After completing this section, you will be able to:

* Understand how decision-making works in Python.
* Write programs using `if`, `if-else`, and `if-elif-else`.
* Use the `match-case` statement effectively.
* Solve real-world problems using nested conditions.
* Write clean, readable, and well-structured conditional logic.
