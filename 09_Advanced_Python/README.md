# Advanced Python

This folder focuses on advanced Python concepts necessary for optimizing performance, managing memory efficiently, and writing production-grade code for AI and Machine Learning systems.

## 📌 Topics to Cover

- **Iterators:** Understanding `iter()` and `next()` mechanisms.
- **Generators:** Memory-efficient data streaming using the `yield` keyword.
- **Decorators:** Modifying or wrapping the behavior of functions dynamically.

## 🚀 AI/ML Context

- **Generators** are crucial for building custom Deep Learning data loaders (like in PyTorch or TensorFlow) to feed large image or text datasets into models batch-by-batch without running out of RAM.
- **Decorators** are used to track training times, log model parameters, or verify user authentication in ML APIs (like FastAPI or Flask).
