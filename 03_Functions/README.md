# ⚙️ Python Functions

This folder introduces Python functions, one of the most important concepts in programming. Functions help organize code into reusable, readable, and maintainable blocks.

It also covers Lambda functions and commonly used functional programming tools such as `map()`, `filter()`, and `reduce()`.

## 📚 Topics Covered

### 📌 User-Defined Functions

* Defining and calling functions
* Parameters and arguments
* Return values
* Default parameters
* Keyword arguments
* `*args`

### 📌 Lambda Functions

* Creating anonymous functions
* Lambda with multiple arguments
* Conditional lambda expressions
* Using lambda inside functions

### 📌 Functional Programming Tools

* **`map()`** – Apply a function to every element of an iterable.
* **`filter()`** – Select elements based on a condition.
* **`reduce()`** – Reduce an iterable to a single value.

## 🎯 Learning Objectives

By completing the programs in this folder, you will learn how to:

* Write reusable functions.
* Pass arguments and return values.
* Simplify code using lambda functions.
* Process collections efficiently with `map()`, `filter()`, and `reduce()`.
* Improve code readability and reusability.

## 💡 Why Learn Functions?

Functions make code reusable, organized, and easier to maintain. They help reduce repetition, improve readability, and simplify problem-solving.

As you progress to Machine Learning and AI, you'll frequently write functions for data preprocessing, feature engineering, model evaluation, and utility tasks.
